# Dev
```shell
git clone https://github.com/Tai7sy/OneDriveFly.git
```

```shell
cd OneDriveFly/php
composer create-project
```

```shell
cd OneDriveFly/nodejs
npm i && npm build
```

The generated files will be output to the `dist` folder
